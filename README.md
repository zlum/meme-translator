# Meme translator

Script to recognize and translate text on images
Uses Tesseract OpenCV and Yandex Ttanslate API

Usage: ./recognition.py --apikey %YOUR YANDEX TRANSLATE API KEY HERE% --padding 0.275 --image images/gag0.webp
